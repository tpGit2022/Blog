> lastupdate 2017/06/05 

项目地址：https://github.com/PhilJay/MPAndroidChart
> 当前最新版本V3.0.2

引入依赖:
项目级别的build.gradle添加如下：
```
allprojects {
    repositories {
        maven { url "https://jitpack.io" }
    }
}
```
在module级别的build.gradle添加依赖：
```
dependencies {
    compile 'com.github.PhilJay:MPAndroidChart:v3.0.2'
}
```
